You can use controller display with N64 if you get an arduino type thing and hook shit up accordingly like that (it's gay if you don't know what you're doing), and yes I believe it works with Hori

You need a GS pro like that yes, version 3.x. You then put the game on top of the GS and put the GS in the console and when turned on it will load the GS code select screen and from there you can choose the game you want to select codes for and also add or edit codes. It will then start up the game. I believe, although not too sure, that you can play NTSC-U games on an NTSC-J console if you have the GS in, obviously it's not useful for speedrunning but if you have an NTSC-U game you wanna casually play then I think it'd work like that. But again I'm not too sure on that.
